### 包管理器安装

#### Homebrew (macOS/Linux)
```bash
brew tap MinatoHikari/tap
brew install gotssh
```

#### Scoop (Windows)
```bash
scoop bucket add gotssh https://github.com/MinatoHikari/scoop-bucket
scoop install gotssh
```

#### AUR (Arch Linux)
```bash
yay -S gotssh
# 或
paru -S gotssh
```
